В этом каталоге размещаются API-контракты сервиса.

## Содержимое

- `openapi.yaml` — актуальный HTTP-контракт сервиса расчета стоимости:
  - настройки ценообразования пользователя;
  - расчет стоимости в рамках чата;
  - история расчетов в чате.

Файл можно использовать для генерации клиентских SDK и документации Swagger UI/Redoc.
